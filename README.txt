DEVELOPMENT ENVIRONMENT: VS Code with JDK 25

DESCRIPTION:
A Java-based GUI application designed with a vintage "As I've Written" (Honkai: Star Rail)
book aesthetic. This application tracks student attendance by capturing 
system-generated timestamps and unique student signatures.

KEY FEATURES IN THIS:
1. THEME: Parchment background (#FAF8F0) with Charcoal ink styling.
2. UPHSD ID SYSTEM: Automatically generates unique e-signatures following 
   the school's ID format (23-1269-XXX) for every entry.
3. DATA PERSISTENCE: All signed entries are saved locally to 
   'attendance_log.txt' in a ledger format.
4. TECHNOLOGY: Utilizes Java Swing for the UI and the latest JDK 25 
   features for reliable execution.

HOW TO RUN:
1. Open the project folder in VS Code.
2. Ensure the Java Extension Pack is active and pointing to JDK 25.
3. Run 'AttendanceTracker.java'.
4. View recorded logs in 'attendance_log.txt'.
